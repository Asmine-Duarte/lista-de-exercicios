A tag table serve para criar uma tabela.

A tag del serve para representar uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto

Exemplo de texto excluído

A tag ins sever para representar um intervalo de texto que foi adicionado a um documento.

Exemplo de texto inserido

A tag sub define o texto subscrito. texto subscrito aparece metade de um personagem abaixo da linha normal, e às vezes é processado em uma fonte menor. Subscrito texto pode ser utilizado para as fórmulas químicas, tal como H 2 O.

Exemplo de texto subscrito

A tag sup define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW

Exemplo de texto sobrescrito

A tag blockquote indica que o texto incluído é uma longa citação.

Exemplo de texto com grande citação

A tag q indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo

Exemplo de pequena citação

A tag abbr representa uma abreviação e opcionalmente fornece uma descrição completa para ela.

Tag abbr

A tag cite representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

https://agenda.museudoholocausto.org.br

A tag bdo é usada para substituir a direcionalidade atual do texto. Isso faz com que a direcionalidade dos personagens seja ignorada em favor da direcionalidade especificada.

A tag code apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código.

A tag code é usada para apresentar uma pequena parte do código.

A tag pre é utilizada para representar texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado.

A tag kbd é usada para definir algum texto como entrada de teclado em um documento.

Ctrl + C é usada para copiar texto.
